Tutorial 1 - 2: Variables

Variables are a very powerful and important part of programing. Variables can store information. This is useful for a variety of uses such as storing the amount of money in a bank acount, solving mathematical equations or storing a players position in games.

In the .code language variables are declared using the variable command. After the variable command comes the name of the variable such as x and then the value such as hello.

The value that is inside of a variable can be accessed and used for another command like this: output (x).

Try it out by typing variable x hello and on the next line output (x).

The output of this program should be "hello"
