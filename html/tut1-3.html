Tutorial 1 - 3: Addition

Another very important thing in programing is the ability to do mathematical operations such as adition.

To do adition in the .code language the add command is used. The command is followed by the output variable in brackets and then two values.

To try this out type
variable out 0
add (out) 36 11 (36 + 11)
output (out)

The output should be 47
